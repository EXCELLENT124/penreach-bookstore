{% trans 'CSV File Format' %}

{% trans 'Your CSV file should include the following columns:' %}

  • title - Book title (required)
  • author - Author name (required)
  • description - Book description
  • price - Price (numeric, e.g., 19.99)
  • isbn - ISBN number (required, unique)
  • publisher - Publisher name
  • publication_date - Publication date (YYYY-MM-DD)
  • pages - Number of pages
  • language - Language (default: English)
  • stock - Stock quantity (default: 0)
  • book_type - Classification: small_books, charts, or big_books
